# Prayer Times Calculator by City
![Contributor Covenant](https://img.shields.io/badge/Contributor%20Covenant-2.0-4baaaa.svg)
![badge](https://img.shields.io/badge/license-MIT-orange)

An app for searching for the prayer(namaz) times for the day by user inputed city. Technologies used: API, Javascript, Bootstrap.

## Description

This application allows the user to search for the prayer times for the current day for any city the user inputs to the search field. A country name is also required to avoide errors in the search.

## Usage 
To use the application, simply input the city you want and the prayer times for that city will be displayed on the page.
